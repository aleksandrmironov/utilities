#Script to backup Github repos


usage:
 ```
-id AWS_ACCESS_KEY_ID
-key AWS_SECRET_ACCESS_KEY
-region AWS_REGION
-o ORGANIZATION
-b BUCKET_NAME
-t TOKEN 
-d DIRECTORY
-a ARCHIVE
-m Enable mirror clone (yes by default)
-e Add datetime ext to the archive filename (yes by default)
-g Additional git params
```
